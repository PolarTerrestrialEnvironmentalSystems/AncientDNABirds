This repository contains the code to reproduce data analyses and figures presented in the manuscript: Sander, M. M., Stoof-Leichsenring, K.R., Liu, S., Shen, W., Lisovski, S., Herzschuh, U. (2025): Sedimentary Metagenomics Reveal Avian Community Transitions From the Last Glacial Maximum to the Holocene published in Ecology and Evolution (https://doi.org/10.1002/ece3.72064). This repository includes the processed datasets for Aves (tmp_LAKENAME), the taxa selection lists (Taxa_selection_...csv) and the scripts for statistical analyses and figure generation (all_lakes.R).

Data availability
All data required to reproduce the analyses and plots in all_lakes.R are provided in this repository (tmp_LAKENAME). These datasets can be reproduced from the raw data and the code by Liu et al. 2026 (https://www.biorxiv.org/content/10.1101/2025.06.23.661215v2) published on https://github.com/sisiliu-research/EnviHoli).

All data required to run ancient_pattern_analysis.R can be reproduced from the raw data and the code provided by von Hippel et al. (2025) (https://zenodo.org/records/15328439) and Çabuk et al. (2025) (https://zenodo.org/records/14810762).
